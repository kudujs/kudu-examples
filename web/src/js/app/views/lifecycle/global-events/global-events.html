Global Events

In this example we show how to subscribe to Kudu lifecycle events. Lifecycle events uses the "lc." prefix, as shown below on line 4. 
var kudu = require("kudu");

kudu.on("lc.init", function(options) {
    console.log("view has been initialized", options);
});

kudu.on("lc.render", function(options) {
    console.log("view has been rendered");
});

kudu.on("lc.fail", function(options) {
    alert("Sorry, an error occurred: ", options.error[0]);
});

Below is the list of global events available and the order in which they are triggered:

  • beforeInit
  • init
  • render
  • complete
  • remove
  • beforeUnrender
  • unrender
    • fail - fail is not part of the order above and can occur during any event
