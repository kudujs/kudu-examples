Binding

binding.html 
<input value="\{{name}}">
<div>Hello \{{name}}</div>
<button on-click="reset()">Set to original value</button>

Line 1 adds a two-way binding to the input value to the name variable through the \{{name}} mustache.

Line 2 adds a one-way binding to the <div> element through the \{{name}} mustache.

Line 3 adds a an on-click listener to the button which calls the method reset(). The reset() method uses the Ractive.set() method to reset the name variale, see line 8 below.

binding.js 
	function createView() {

			var view = new template({
				
				data: {"name": "World!"},
				
				reset: function() {
					this.set("name", "World!");
				}
			});
			return view;
		}
});
