Redirecting to another view

Sometimes you want to redirect to a different view, if certain validations are not in place, or certain permissions are missing.

When creating a view with Ctroller.onInit, it is possible to navigate to a different view, through kudu.go(...), and return null. By returning null from Controller.onInit, kudu will stop the initialization process of that view.

For example:

that.onInit = function(options) {

    // Check if typeId parameter is present. This view requires a typeId to function
    if (options.typeId == null) {

        // redirect to a different view to pick a type
        kudu.go( {ctrl: CarTypeController} )
        return null;
    }

    // typeId parameter is present so we create the view for this controller and return it
    var view = createView();
    return view;
    
}
