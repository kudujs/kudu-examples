Basic Component

In this demo, we show how to create a basic component. While components are really a Ractive feature, we will be showcasing it here as well.

In this demo we create a very simple component: a <div> tag with a label and a click action that alerts the user.

Below is the component in action!

Below is the code to create the component: 

function createComponent() {
            
			 var comp = ractive.extend({
                template: '<div on-click="activate()"> \{{message}} </div>',
                activate: function() {
                    alert("Active!");
                },
                data: {
                    // A default message if none is provided
                    message: 'No message specified, using the default'
                }
            });
};

Looking at the code above, you'll note that a Ractive component is just a normal Ractive instance. Line 4 shows the template, line 5 a method to invoke, and line 8 a data object.

What turns a ractive instance into a component is when it referenced inside another ractive instance, as show below:

function createView() {
    var comp = createComponent():
    var view = new template({
        components: {
            simple: comp
        }
    });
};

Line 4 registers the ractive components, as key/value pairs.

Line 5 we register a component called simple which references the comp instance. simple becomes a new tag that we can reference in our template: 

<simple message='Click to activate!'/>

Line 1 shows how to specify the component in our html markup. The name of the tag is the same used when registering our component.

We can set the component message as an attribute.
