Controller Events

Controllers provide a number of lifecycle hooks for you to take action in. To add a hook, simply declare the function on your controller: controller.onInit as shown below. 
function myController() {

		var that = {};

		that.onInit = function (options) {
        console.log("onInit called", options);
        return new template();
    };

    that.onRender = function(options) {
        console.log("onRender called", options);
    }

    that.onComplete = function(options) {
        console.log("onComplete called", options);
    }

    that.onRemove = function(options) {
        console.log("onRemove called", options);
    }
}

Here is the list of controller lifecycle events available and the order in which they are invoked:

  • onInit
  • onRender
  • onComplete
  • onRemove
  • onUnrender
