Multiple Components

This demo shows how to switch between multiple components.

It is rather simple, as Ractive does all the heavy lifting.

Below are two components in action. Click the buttons to switch between the components.

{{#if componentName == 'A'}} {{else}} {{/if}}

All we do is put the two components in a conditional mustacache and Ractive will switch between the two components.

Below is the Javascript:

 function createView(data) {
            
            var compA = ractive.extend({
                template: '<div class="comp">I am component A</div>'
            });
            
            var compB = ractive.extend({
                template: '<div class="comp compB">I am component B</div>'
            });

            var view = new template({

                components: {
                    compA: compA,
                    compB: compB
                },
                
                data: {
                    componentName: 'A'
                }
            });

            return view;
        }

We create two components, compA and compB in lines 3 & 7 respectively.

In lines 14 & 15 we register the two components with the view.

Line 19 sets the componentName data variable to 'A'.

Below is the html template:

\{{#if componentName == 'A'}}
    <compA/>
\{{else}}
    <compB/>
\{{/if}}

<button on-click="set('componentName', 'A')">Render Component A</button>
            
<button on-click="set('componentName', 'B')">Render Component B</button>

In line 1 we use a conditional statement to check the value of componentName.

Line 2 we render compA otherwise in line 4 we render compB.

Line 7 & 9 shows two buttons, that when clicked sets the value of the componentName variable to either 'A' or 'B'.

Changing the value of the componentName variable will render either compA or compB.
