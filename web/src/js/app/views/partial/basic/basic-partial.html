Basic Partial

Partials are template snippets that can be embedded into other templates. See here for details.

Below is an example partial

{{>hello}}

Below is the javascript code:

 var helloPartial = require("rv!./partial");

function createView() {

    var view = new template({				
        partials: {hello: helloPartial}
    });
    return view;
}

In line 1 we require a partial called helloPartial. Note: we use the rv! plugin instead of the rvc! plugin since we simply want the template, not a component. The rv! plugin compiles the template ready for Ractive to comsume.

Line 6 we register the hello partial which references the helloPartial.

Next we show the template to render a partial:

 \{{>hello}}

In line 1 we use > character inside a mustache to reference the hello partial.

Finally we show the actual partial below:

<h3 class="partial">
    Hi, I am a partial!
</h3>
