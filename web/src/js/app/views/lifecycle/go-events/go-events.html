Kudu.go() Events

Kudu.go() provide a number of lifecycle hooks for you to take action in. To add a hook, simply declare the function on the options passed to the go() function: options.onInit as shown below. 
var kudu = require("kudu");
var $ = require("jquery");
var myController = require("app/pages/myController");

...

kudu.go({ctrl: myController,
					
    init: function (options) {
    	console.log("init", options, options.ctrl);
    },
					
    remove: function (options) {
        console.log("remove", options, options.ctrl);
    }
});

Here is the list of controller lifecycle events available and the order in which they are invoked:

  • onInit
  • onRender
  • onComplete
  • onRemove
  • onUnrender

Note: open the browser console and click the Reload view button below, to see the order of events called.
